# App Brief

Build a small appointment scheduling app for local salons.

## What This App Is
- A web app that lets customers book available appointment slots.

## Target Users
- Salon staff who need a simple schedule.
- Customers booking from a phone.

## Constraints
- Mobile-friendly first.
- Start with a single-location workflow.
- Prefer a simple stack over premature complexity.

## Success Criteria
- Staff can publish availability.
- Customers can book and cancel appointments.
- The first usable version can be tested locally by one shop.

## Current Focus
- Land the first booking flow end to end.
